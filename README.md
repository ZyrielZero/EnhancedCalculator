# Enhanced Calculator

A command-line REPL calculator built with object-oriented design and five design
patterns (Strategy, Factory, Observer, Memento, Facade). Supports addition,
subtraction, multiplication, division, power, and root, persists history to CSV
with pandas, configured through environment variables, with 100% test coverage
enforced through pytest and GitHub Actions.
